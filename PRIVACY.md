# 隐私政策 / Privacy Policy

最后更新：2026年5月25日

## 概述
AI Prompt 管家（以下简称"本扩展"）尊重并保护您的隐私。本扩展**不会收集、传输或存储任何个人数据**。

## 数据收集
本扩展 **不会收集** 任何类型的个人数据，包括但不限于：
- 浏览历史
- 表单输入内容
- 网页内容
- 位置信息
- 个人身份信息

## 数据存储
本扩展使用 Chrome Storage Sync API 存储以下数据：
- 用户创建的提示词（Prompt）数据

所有这些数据仅存储在您的 Chrome 浏览器本地，并通过您的 Google 账号在您自己的设备之间同步。**没有任何数据被发送到外部服务器**，包括我们的服务器。

## 数据共享
本扩展 **不会** 与任何第三方共享任何数据。

## 权限说明
本扩展请求以下 Chrome 权限：
- `storage` — 用于在本地存储和同步您的提示词数据
- `clipboardWrite` — 用于将提示词内容复制到剪贴板

## 第三方服务
本扩展不集成任何第三方分析、广告或追踪服务。

# Privacy Policy (English)

Last updated: May 25, 2026

## Overview
AI Prompt Manager (the "Extension") respects your privacy. This Extension **does not collect, transmit, or store any personal data**.

## Data Collection
This Extension **does not collect** any type of personal data, including but not limited to:
- Browsing history
- Form inputs
- Web page content
- Location data
- Personally identifiable information

## Data Storage
This Extension uses the Chrome Storage Sync API to store:
- User-created prompt data

All data is stored locally in your Chrome browser and synced across your own devices via your Google account. **No data is sent to any external server**, including ours.

## Data Sharing
This Extension **does not** share any data with any third party.

## Permissions
This Extension requests the following Chrome permissions:
- `storage` — to store and sync your prompt data locally
- `clipboardWrite` — to copy prompt content to your clipboard

## Third-Party Services
This Extension does not integrate any third-party analytics, advertising, or tracking services.

For privacy-related questions, contact: [your-email@example.com]
